Random Password Generator

The random password generator was my second project I had coded. Using what I had learn from my first project, I had a relatively easy time coding the random password generator.

Although it was easier to code my second project. I still had a few problems while coding the password generator as there were many instances where the code wouldn't run in the terminal due to syntax errors and undeclared variables. However, after carefully looking through the code I managed to find out certain errors such as the wrong punctuation and forgetting to put a bracket between variables. Eventually, I was able to get the code to run.

Source Code

import random

chars = "abcdefghijklmnopqrstuvwxyzABCDEFHIGJKLMNOPQRSTUVWXYZ1234567890!@#$%^&*(')"
										
while 1:
password_len = int(input("what length would you want your password to be?: "))
 password_count = int(input("how many passwords would you like?: "))
 for x in range(0, password_count):
 password = ""
for x in range(0, password_len):
 password_char = random.choice(chars)
 password = password + password_char
 print("Here is your password :", password)
